Reverse String.

- Use the Slice method to Reverse String.

```py
def my_function(x):
  return x[::-1]
```

```py
mytxt = my_function("I wonder how this text looks like backwards")
print(mytxt)
# output : sdrawkcab ekil skool txet siht woh rednow I
```
